Introduction

NEBA makes an AEM developer's life easier. It provides a model for the current resource as a @ResourceModel to the view, automatically creates AdapterFactories for all resource models. This reduces the amount of boilerplate code that is necessary to write AEM application. And less code is a good thing!

NEBA also enables the use of Spring and Spring MVC within Adobe AEM, providing rich IOC capabilities and thereby provides enterprise support.

NEBA is an open software project released under the permissive Apache license. We welcome contributions, comments and opinions!
