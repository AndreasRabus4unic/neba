Quick facts

NEBA enables the development of clean, high-performing and easy to change solutions on Apache Sling-based applications such as Adobe AEM™. NEBA integrates the Spring framework into Apache Sling, exclusively using the standardized reference implementation of the OSGi Compendium R4.2 specification.

NEBA is 100% open source and published under the Apache license 2.0. It runs on standalone Sling and Adobe AEM™ 5.3 - 6.x using Java 1.6+.
